Recommend the site or the current page to a friend by mail.

== Description ==

This plugin adds a simple widget that allows you to display an email form and a submit button.

* You have the options to choose to recommend the site or the current page.
* The mail subject and content can be changed.
* You can limit the number of recommendation sended by user a day.

The widget is only displayed if the user is connected.

== Installation ==

1. Upload the 'recommend-by-mail-widget' directory to the '/wp-content/plugins/' directory ;
2. Activate the plugin through the 'Plugins' menu in WordPress ;
3. Navigate to the 'Widgets' page found under the 'Appearance' menu item ;
4. Drag 'Recommend by mail widget' to the target widget area and choose your options.
